# Site Web GEM

Site web du **Groupe d'Entraide Mutuelle (GEM)**, un lieu d'entraide entre pairs créé autour du vécu et des besoins des personnes autistes. Conçu par des autistes, pour des autistes.

## Structure du projet

```
├── index.html        # Page principale
├── style.css         # Styles
├── main.js           # Scripts
├── favicon.ico       # Icône du site
├── AGORA_LOGO_RGB.svg
├── Mascotte_bonjour.png
├── Mascotte_joueur.png
├── Mascotte_libraire.png
└── Contact_Mascotte.png
```

## Technologies

- HTML5, CSS3, JavaScript (vanilla)
- Police [Lexend](https://fonts.google.com/specimen/Lexend) via Google Fonts
- Conçu avec une attention particulière à l'accessibilité

## Lancer le site

Ouvrir `index.html` directement dans un navigateur, ou utiliser une extension comme **Live Server** dans VS Code.
